# ⚡ Velocity Dash

**Velocity Dash** is a high-speed, browser-based action game where every millisecond counts. Dodge obstacles, maintain your momentum, and see how long you can survive the dash!

## 🚀 Game Objective
The goal is simple: **Don't stop moving.** Navigate through a series of increasingly difficult levels where the speed ramps up the longer you stay alive. 

### Features:
* 🏃 **Instant Restart:** No loading screens, just pure action.
* ⚡ **Speed Scaling:** The game gets faster as your score increases.
* ⌨️ **Keyboard Optimized:** Tight controls designed for desktop play.

## 🛠️ Built With
* **HTML5 Canvas** - For rendering high-speed graphics.
* **JavaScript** - For the game logic and physics.
* **CSS3** - For the sleek, neon-inspired UI.
